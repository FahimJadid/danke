### [SAP](https://www.sap.com/)

System application and products for real time data processing or\
System application and products in real time data processing

- SAP is an ERP product
- SAP introduced by SAP AG in 1972 at Waldorf, Germany
- It is German based product

### ERP: E:Enterprise R:Resource P:Planning

It’s purpose is to manage any company functions:
- It's purpose is to manage any organization/company functions.
- It is a way to integrate the data and processes of an organization into one single system

### Features of SAP:
- It is an integration of all functions in to one common software
- It is a Multilingual software
- Support multiple currencies
- It is a user-based license agreement (requires username and password)

### ABAP/4 :
- ABAP - Advanced Business Application Programming. It is a programming language for SAP. 
- It is a 4th generation language
- ABAP lies on application Layer

### SAP Netweaver:
SAP Netweaver is an integrated technology platform which can support various SAP applications. (Applications Within a Container Application)
- SAP ERP (Enterprise Resource Planning)
- SAP CRM (Customer relationship Management)
- SAP SRM (Supply Relationship Management)
- SAP PI (Process Integration)

### Modules of SAP:
SAP divided the modules broadly in to 2 categories:
1. Functional Modules ⇒
    - SAP SD (Sales and Distribution)
    - SAP MM (Material Management)
    - SAP PS (Project System)
    - SAP FICO (Finance and Controlling) 
    - SAP HR (Human Resources) ⇒ HCM (Human Capital Management)
    - SAP WM (Warehouse Management)
2. Technical Modules ⇒
    - SAP ABAP (Programming language of SAP)
    - SAP BASIS (Business Application Software Integrated Solution. It is for monitoring and administration of SAP systems.)
    - SAP BIBO (Business Intelligence, Business Objects) ⇒ Reporting tool (It is used to analyze and presents the data in the form of reports, dashboards etc.
    - CRM - Customer Relationship Management
    - PI - Process Integration
)

### SAP R/3 Architecture:
1. (3 - 3 tier architecture, R - Real ) - SAP is a real time architecture
Layers: 
    - Presentation Layer (User Interaction Layer)
        1. Presentation Layer is also called as user interaction Layer
        2. It acts as an interface between end user and SAP systems
        3. It collects user input and creates process request

    - Application Layer (Writing the programs and applications)
        1. It acts as an interface between Presentation layer and database layer
        2. Application layer is responsible for writing the programs and applications
        3. It uses the application logic of program to collect and process request

    - Database Layer (To Store and retrieve the data)
        1. Databases(oracle, DB2 etc) reside on the database layer
        2. It stores and retrieves the data
